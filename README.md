# 📱 YUIMEI LIFE — PWA Version

有意義生活ダッシュボード / Life Optimization App

## セットアップ方法（3ステップ）

### ステップ 1️⃣ Netlifyにアクセス
https://netlify.com にアクセス

### ステップ 2️⃣ ファイルをアップロード
1. Netlifyのトップページで「Drag and drop your site folder here」という枠を探す
2. このフォルダ内のファイル全て（index.html, manifest.json, sw.js）を**ドラッグ&ドロップ**
3. 自動でURLが生成される（例: `https://yuimei-abc123.netlify.app`）

### ステップ 3️⃣ Androidスマホでホーム画面に追加
1. 生成されたURLをAndroidのChromeで開く
2. 右上の「⋮」メニュー → 「アプリをインストール」（または「ホーム画面に追加」）
3. 完了！ホーム画面にアイコンが追加されます

## ファイル説明

- **index.html** — メインアプリケーション（全てのコード）
- **manifest.json** — PWA設定（アイコン、名前、動作など）
- **sw.js** — Service Worker（オフライン対応）

## 機能

✅ 勉強時間管理（科目・参考書別）  
✅ タイマー機能（全画面、ポモドーロサイクル対応）  
✅ デイリーチェックリスト（カスタマイズ可能）  
✅ 健康ログ（有酸素・筋トレ・睡眠など）  
✅ 日記機能（日付付き）  
✅ 知識フィード（国際政治・金融・英語など）  
✅ 過去の記録閲覧  
✅ 有意義生活指数（0〜100点）  
✅ オフライン対応  
✅ データはスマホ内に保存
